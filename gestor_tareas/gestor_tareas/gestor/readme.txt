buenas noches, aqui voy a describir el trabajo, el trabajo es un gestor de tareas hecho en python, contiene una lista de opciones para el usuario
donde puede agregar tareas con una leve descripcion, ver las listas de las tareas existentes,Filtrar tareas por su estado (Pendiente o Completada)
marcar tareas como completadas para un mejor seguimiento, eliminar tareas del sistema según su ID, interfaz interactiva con un menú de opciones fácil de usar.
Ademas contiene una opcion de error al final del ciclo por si la persona elije alguna opcion no correspondiente en la lista.

su funcionamiento seria:

El usuario selecciona una opción del menú.

Puede agregar nuevas tareas, ver la lista de tareas existentes, marcar tareas como completadas o eliminarlas.

El sistema asigna un ID autoincremental a cada tarea para una mejor gestión.

Las tareas se almacenan temporalmente en una lista dentro de la clase GestorTareas.

El programa se ejecuta en un bucle hasta que el usuario decide salir.

es un sistema simple hecho en python para que sea visible en consola, es facil de gestionar y ideal para quienes buscan un gestor de tareas sencillo.
